# UT4-Pedido-de-Restaurante

## Autores : Izan y Iván

## Descripción
Sistema de gestion de pedidos para restaurante desarrollado en Java. Permite registrar pedidos de clientes con hasta 5 articulos, calcular el precio total automaticamente y gestionar el estado del pedido.

## Caracteristicas
- Generación automática de IDs únicos para cada pedido
- Gestión de hasta 5 artículos por pedido
- Cálculo automático del precio total
- Control de estados del pedido (EN_PREPARACION, LISTO_PARA_ENTREGAR, ENTREGADO)
- Validaciones para evitar retroceso de estados
- Validaciones para limite de artículos

## Como Ejecutar 
Desde la terminal :
java main

Desde un IDE por ejemplo en nuestro caso seria en Visual Studio Code:
1. Abre el proyecto en tu IDE
2. Ejecuta la clase Main.java
